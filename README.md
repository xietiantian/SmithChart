# SmithChart
一个史密斯园图应用。能完成简单的阻抗匹配。

实现了Smith圆图中触控取阻抗点，拖拽修改阻抗点，等电阻圆、等电抗圆、等反射系数圆、驻波比圆的绘制，圆图的放缩、拖拽，输入阻抗、反射系数的计算，展示随着观察点的距离到负载距离变化时圆图中各点、圆的变化等功能。

继承view类编写了放缩图片类ZoomImageView，并重载其绘制方法（加入了点、辅助圆的绘制）。
